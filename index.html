Research Focus

From data flywheels to interactive agency

01

GUI Agent Trajectory Pipelines

Constructing high-quality human demonstrations and automated rollouts across desktop, web, and game environments, including action schemas, validation, filtering, and conversion.

02

SFT for Actionable Agents

Turning multimodal trajectories into supervision for perception, planning, grounding, and robust execution over screenshots, UI states, and structured action spaces.

03

Game-play Reinforcement Learning

Training agents with environment feedback, task progress signals, and game-play rewards so that models can improve through interaction rather than static imitation alone.
